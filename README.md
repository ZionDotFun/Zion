## 🚀 Quick Start
Create a Virtual Environment:
```
python -m venv venv
```
Activate the Virtual Environment:
```
source venv/bin/activate
```
Install ZION:
```
pip install zionpy

```

Verify Installation:
```
import zionpy
print(zionpy.__version__)  \
```
